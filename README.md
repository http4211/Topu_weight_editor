Topu_weight_editor - グリッドNパネル (v0.2.11)

インストール:
- Blender > 編集 > プリファレンス > アドオン > インストール
- このZIPを選択してインストールし、アドオンを有効化

使い方:
- 3Dビュー > Nパネル > 「TPWE」タブ
- 「アクティブを対象に」または「対象メッシュ」で編集対象を指定
- 編集モード（推奨）で頂点を選択
- 「リアルタイム更新」ON: 頂点選択が変わるたびにグリッドが自動更新
- 「リアルタイム更新」OFF: 「手動更新」で更新

グリッド:
- 列ヘッダー（グループ名）をクリックすると、その列が「選択列」になります
- セルを直接編集すると Abs 代入（ロック列は編集不可）
- 「選択列に適用 / 表示列に適用」で一括編集（Abs / Add / Add%）

その他:
- 「正規化」ON: 編集後に合計が1.0になるよう調整（ロック列は保持）
- 「合計>1を許可」ON: 合計1.0超を許可（正規化をスキップ）
- 「影響数上限」+「影響数制限を適用」で、頂点あたりの影響数を整理
- 「小数点の整理」: ウェイトを指定桁に丸め、合計が変わらないように調整（ロック/無視列は除外）
- 高速化したい場合は「選択頂点のみ表示」をON推奨

v0.2.11 変更点（パフォーマンス）:
- リアルタイム更新の選択監視を軽量化（毎回の全選択スキャンを回避）
- 列の「使用判定」を表示中の行（ページ）に限定して、再構築を高速化
